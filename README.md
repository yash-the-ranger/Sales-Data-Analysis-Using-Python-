# Sales Data Analysis Using Python

## Project Objective

The objective of this project is to analyze sales data using Python. The project includes data cleaning, revenue analysis, category-wise performance, region-wise performance, top product identification, and visualization of sales trends.

## Project Workflow

1. Import required libraries
2. Load the sales dataset
3. Check dataset structure
4. Handle missing values
5. Remove duplicate records
6. Convert date columns
7. Create new columns
8. Perform revenue analysis
9. Analyze region-wise and category-wise performance
10. Create visualizations
11. Write business insights
