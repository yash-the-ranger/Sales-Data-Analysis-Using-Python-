# Sales Data Analysis Using Python

## Project Objective

The objective of this project is to analyze sales data using Python. The project includes data cleaning, data transformation, revenue analysis, category-wise performance, region-wise performance, product analysis, monthly sales trend analysis, and data visualization.

## Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Dataset Overview

The dataset contains sales transaction details such as:

- Order ID
- Order Date
- Region
- Salesperson
- Category
- Product
- Units
- Unit Price
- Revenue
- Customer Type

## Project Workflow

1. Imported required Python libraries
2. Loaded the sales dataset
3. Checked dataset structure, rows, columns, and data types
4. Identified and handled missing values
5. Removed duplicate records
6. Standardized column names
7. Converted date columns into proper datetime format
8. Created new columns such as Month and Revenue Category
9. Performed revenue analysis
10. Analyzed category-wise, region-wise, and product-wise performance
11. Created visualizations using Matplotlib, Seaborn, and Plotly
12. Wrote business insights and conclusion

## Data Cleaning

In this step, I checked missing values, duplicate records, data types, and column names. I removed duplicate records, standardized column names, converted the order date column into proper datetime format, and converted revenue into numeric format.

## Data Transformation

In this step, I created new columns from the cleaned dataset. I extracted the month from the order date for monthly trend analysis and created a revenue category column to classify revenue values as High Revenue, Medium Revenue, or Low Revenue.

## Analysis Performed

- Total revenue analysis
- Average revenue analysis
- Highest and lowest revenue analysis
- Revenue by region
- Revenue by category
- Top 5 products by revenue
- Salesperson performance analysis
- Monthly revenue trend analysis
- Region and category pivot analysis

## Visualizations Created

- Revenue by Region bar chart
- Revenue by Category chart
- Monthly Revenue Trend line chart
- Revenue Share by Category pie chart
- Top 5 Products by Revenue chart
- Correlation heatmap

## Key Business Insights

1. Region-wise analysis helps identify the best-performing and low-performing regions.
2. Category-wise analysis shows which product category contributes more to total revenue.
3. Product analysis helps identify top revenue-generating products.
4. Monthly trend analysis helps understand peak and low-performing sales periods.
5. Visualizations make it easier to compare revenue patterns across region, category, and product.

## Conclusion

This project analyzed sales data using Python. Pandas was used for data cleaning, transformation, and analysis. Matplotlib, Seaborn, and Plotly were used for visualization. The project helped identify revenue trends, top-performing regions, best-selling categories, top products, and useful business insights.
